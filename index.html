Local LAN (UDP) — direct, no internet needed

        
Send the JSON payload to your device IP on UDP port 4003.

        

        
Example using netcat (Linux/Mac):

        
echo '<JSON>' | nc -w1 -u 192.168.x.x 4003

        
Example using Python:

        
import socket, json
s = socket.socket(socket.AF_INET, socket.SOCK_DGRAM)
s.sendto(json.dumps(payload).encode(), ('192.168.x.x', 4003))

        

        
Browser limitation: Browsers cannot send raw UDP. Use the exported JSON with the script above, or run a local WebSocket-to-UDP bridge.
